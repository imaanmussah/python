# IT Helpdesk Ticket Registration System

## Purpose
This application allows students to register IT support tickets and automatically assigns a technician based on the selected priority level.

## Tech Stack
- Python 3

## How to Use
1. Run `main.py`.
2. Enter the student information.
3. Select a priority (High, Medium or Low).
4. The system assigns a technician automatically.
5. The completed helpdesk ticket is displayed.

- Medium → Siti
